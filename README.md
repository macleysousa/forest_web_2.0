
# forest_web_2.0

Este projeto utiliza o meta-framework https://nextjs.org/

## Pré-Requisitos

1. Node.js v18
2. NPM v9

## Instalação

```
npm install
cp .env.example .env.local
```

## Configuração

Altere o `.env.local` com os valores corretos para seu ambiente

## Inicialização

### Modo de desenvolvimento

```
npm run dev
```
### Modo de produção

```
npm run build
npm run start
```
